Use "ESPlorer.exe" to start ESPlorer with increased font-size.

(https://github.com/4refr0nt/ESPlorer/issues/95#issuecomment-1768306226)


First create "ESPlorer.jar" as described on the webpage: https://github.com/4refr0nt/ESPlorer

run "ESPlorer.exe" or "ESPlorer32.exe" to start ESPlorer then.


ESPlorer.exe is a simple start script created with Autohotkey V1.
Configuration-file: "loader.ini"


Compile Loader from source ("loader.ahk"):
(needs an installed Autohotkey version 1.xx)

run "compileESPlorer.bat"

Loader homepage:
https://github.com/jvr-ks/loader
